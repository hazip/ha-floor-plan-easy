Adding the Floor Plan editor without Browser Mod

The recommended way to open the editor is via the Browser Mod integration — with it installed, an “Open editor” button appears in this card's configuration panel.

If you don't want to install Browser Mod, you can place the editor directly on a dashboard as its own card. It is fully functional on its own: it starts with an empty floor and has a toolbar for loading and saving floors.

Steps

  1. Open your dashboard and enter Edit dashboard mode.
  2. Create a new view (tab) for the editor — this keeps it separate from your normal cards.
  3. In that view choose Add card, scroll to the bottom, and pick Manual (the YAML card).
  4. Paste the following configuration:
type: custom:floor-plan-easy-editor
  1. Save the card. The full editor now appears on that tab.
  2. Use the editor's toolbar to load an existing floor or to save your work to the backend.
Why isn't the editor in the card picker?
Only the viewer card (custom:floor-plan-easy) is registered in the visual card picker, to avoid confusing users with two similar-looking cards. The editor is added manually via the Manual/YAML card as shown above.
